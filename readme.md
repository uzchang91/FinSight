# FinSight

FinSight is a fintech + edtech web platform designed to help beginners learn how the stock market works through interactive learning, gamification, and progress tracking.

## Overview

FinSight turns financial education into a guided learning experience. Users earn points and achievements as they learn about stock fundamentals, charts, and market concepts.

The goal is to make financial literacy more approachable for beginners by combining:

- Educational content
- Gamification systems
- Progress tracking
- Interactive dashboards

## Core Features

### Learning System
Users can learn stock market fundamentals through structured lessons.

Topics include:
- What stocks are
- How markets work
- Reading stock charts
- Understanding indicators

### Gamified Progress
Learning activities reward users with:

- Points
- Achievements
- Progress tracking

This system encourages consistent learning and engagement.

### Dashboard
A central dashboard allows users to:

- Track their learning progress
- View earned achievements
- Monitor learning milestones

### Authentication
Supports social login for a smooth onboarding experience.

Planned providers:
- Kakao Login
- Google Login

## Getting Started

Clone the repository:

```
git clone https://github.com/your-username/FinSight.git
```

Install dependencies:

```
cd frontend
npm install
```

Run the development server:

```
npm run dev
```
